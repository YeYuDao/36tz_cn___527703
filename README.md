<br/></br>### 课程介绍:
词向量（Word embedding）是深入学习技术在自然语言处理中应用的基础，因此掌握好词向量是学习深度学习技术在自然语言处理用应用的重要环节。本课程从One-hot编码开始，word2vec、fasttext到glove讲解词向量技术的方方面面，每个技术点环节都有相应的小案例，以更加巩固相应知识。
